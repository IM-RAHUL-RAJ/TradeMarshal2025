# Updated Environment Configuration for Ingress Setup

## For Path-based Routing (ingress.yaml)
# Update your Angular environment files to use:
export const environment = {
  production: true,
  apiBaseUrl: 'http://<ALB_DNS>/api/'  // Will be provided after ingress deployment
};

## For Host-based Routing (ingress-hosts.yaml)
# Update your Angular environment files to use:
export const environment = {
  production: true,
  apiBaseUrl: 'http://api.trademarshals.com/'  // You'll need to configure DNS
};
